# 🎵 Super Search Lossless Cutter
**Oleh: Abriel Wiradika Utama**

Skrip otomatis berbasis Windows Batch untuk mencari, memilih, dan memotong file audio secara presisi tanpa mengurangi kualitas (Lossless) menggunakan *engine* FFmpeg.

---

### ✨ Fitur Unggulan
* **Smart Search:** Mencari file audio di seluruh folder dan subfolder hanya dengan kata kunci.
* **Auto-Deployment FFmpeg:** Jika FFmpeg tidak ditemukan di sistem, skrip akan mengunduhnya secara otomatis langsung dari GitHub resmi.
* **Lossless Cutting:** Memotong audio tanpa proses encoding ulang, sehingga kualitas tetap 100% sama dan proses sangat cepat.
* **Visual Interface:** Dilengkapi dengan petunjuk format waktu dan antarmuka yang bersih.

---

### 🚀 Cara Penggunaan
1.  Unduh file `music cutter.bat`.
2.  Letakkan di folder tempat koleksi audio kamu berada.
3.  Klik dua kali pada `music cutter.bat`.
4.  Ikuti instruksi di layar:
    * Masukkan kata kunci nama file.
    * Pilih nomor file jika ditemukan lebih dari satu.
    * Masukkan waktu mulai dan selesai (Contoh: `01:30` atau `00:45`).
5.  Hasil potongan akan muncul di folder yang sama dengan awalan `CUT_`.

---

### 🛠️ Persyaratan Sistem
* Windows OS (CMD/PowerShell).
* Koneksi internet (hanya saat pertama kali dijalankan untuk mengunduh FFmpeg jika belum ada).
